### 파이썬 패키지 환경을 쉽게 관리할 수 있도록 하는 requirements.txt
<br>
- 자신이 구성한 환경을 requirements.txt로 만드는 명령어

```bash
$ pip freeze > requirements.txt
```
<br>
- requirements.txt를 통해 지정한 환경에 설치하는 명령어

```bash
$ pip install -r requirements.txt
```
